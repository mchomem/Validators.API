# DocBrNet.API

The DocBrNet API is a project with functionality to validate and generate unique identifier data, such as CPF and CNPJ.
To calculate the alphanumeric CNPJ, the following document must be accessed:
[gov.br](https://www.gov.br/receitafederal/pt-br/centrais-de-conteudo/publicacoes/documentos-tecnicos/cnpj/manual-dv-cnpj.pdf/view).
